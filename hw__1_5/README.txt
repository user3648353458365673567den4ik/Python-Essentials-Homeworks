В целях придерживания принципа DRY (don't repeat yourself), я вынес метод to_json в метакласс, чтобы все классы могли сериализовываться в json без доп. кода.
Также благодаря метаклассу функция generate_serialization_code будет точно сериализовывать все классы которые нам могут понадобятся.
С помощью библиотеки Faker была добавлена возможность случайной генерации людей.
